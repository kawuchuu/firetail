# Running / building
You can either use the executables provided or build it yourself.

## Run
  1. `git clone https://github.com/projsh/firetail.git`
  2. `cd firetail`
  3. `yarn install`
  4. `yarn start`

## Build
  1. `git clone https://github.com/projsh/firetail.git`
  2. `cd firetail`
  3. `yarn install`
  4. `yarn global add electron-builder`
  5. `yarn release`
